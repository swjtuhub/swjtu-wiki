交通路线

# 交通路线

# 犀浦校区

乘坐火车在 成都东站 下车的新生，可乘坐 地铁7号线外环 到 西南交大 换乘 地铁6号线 到 兴业北街（西门附近）。
或乘坐 地铁2号线 到 牛王庙 换乘地铁6号线到 兴业北街（西门附近）。
或乘坐 地铁2号线 到 天河路 换乘 有轨电车蓉2号线支线（仁和方向） 到 交大犀浦校区（西门附近）。

乘坐火车在 成都站（火车北站） 下车的新生，可乘坐 地铁7号线外环 到 西南交大 换乘 地铁6号线 到 兴业北街（西门附近）。
或乘坐 地铁1号线 到 人民北路 换乘 地铁6号线 到 兴业北街（西门附近）。

乘坐飞机到 双流国际机场 的新生，可乘坐 地铁10号线 到 太平园 换乘 地铁7号线内环 到 西南交大 换乘 地铁6号线 到 兴业北街（西门附近）。
或乘坐 地铁10号线 到 太平园 换乘 地铁3号线 到 前锋路 换乘 地铁6号线 到 兴业北街（西门附近）。

乘坐飞机到 天府国际机场 的新生，可乘坐 地铁18号线 到 火车南站 换乘 地铁1号线 到 人民北路 换乘 地铁6号线 到 兴业北街（西门附近）。
或乘坐 地铁18号线 到 火车南站 换乘 地铁7号线内环 到 西南交大 换乘 地铁6号线 到 兴业北街（西门附近）。

注意

由于犀浦校区西门正在施工改造，请乘坐地铁的同学在 兴业北街 地铁站 C 口出站，往前行 100 米，从 西南一门 入校。乘出租车、私家车到校的，请直接在犀浦校区南门下车，走南门入校。

# 九里校区

乘坐火车在 成都东站 或 成都站（火车北站） 下车的新生，可乘坐 地铁7号线外环 到 九里堤（南门附近）或 西南交大（西门附近）。

乘坐飞机到 双流机场 的新生，可乘坐 地铁10号线 到 太平园 换乘 地铁7号线内环 到 九里堤（南门附近）或 西南交大（西门附近）。

乘坐飞机到 天府国际机场 的新生，可乘坐 地铁18号线 到 火车南站 换乘 地铁1号线 到 火车北站 换乘 地铁7号线外环 到 九里堤（南门附近）或 西南交大（西门附近）。
或乘坐 地铁18号线 到 火车南站 换乘 地铁7号线内环 到 九里堤（南门附近）或 西南交大（西门附近）。

* 九里堤地铁站 D 口出站，步行至九里校区南门；西南交大地铁站 F 口出站，步行至九里校区西门。
